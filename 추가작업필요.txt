KorPark 사이트 — 방문자 관점 이미지·링크 개선 제안
전체 섹션을 직접 분석한 결과를 방문자 입장에서 정리했습니다. 현재 상태와 추가되면 좋을 요소를 섹션별로 구체적으로 제안합니다.

🏠 Hero (첫 화면)
현재: 배경 사진 1장(고흥 바다 항공샷) + "프로그램 보기", "사전 문의하기" 버튼 2개
추가하면 좋을 것들

영상 배경 또는 짧은 하이라이트 영상 링크 — 드론으로 촬영한 고흥 상공·캠핑장 영상은 첫인상에서 압도적 임팩트를 줍니다. 유튜브 영상이 있다면 배경 영상이나 팝업 재생 버튼으로 연결하면 좋습니다.
SNS 링크 아이콘 (인스타그램, 유튜브, 블로그) — 현재 푸터·히어로 어디에도 SNS 링크가 없습니다. 방문자가 비주얼 콘텐츠를 더 보고 싶을 때 갈 곳이 없습니다.
전화번호 클릭 링크 (tel:) — 모바일 방문자가 바로 전화할 수 있도록 Hero 하단에 연락처 클릭 링크가 있으면 편리합니다.


🏫 소개 (About) 섹션
현재: 리모델링 컨셉 이미지 1장 + 현재 모습 1장, 링크 없음
추가하면 좋을 것들

Zone별 사진 각각 1장씩 — Zone A(캠핑), Zone B(교육관), Zone C(루프탑 드론장)를 설명하는데 글만 있고 사진이 없습니다. 각 Zone의 실제 공간 사진이 있으면 현장감이 생깁니다.
갤러리 이미지 클릭 시 라이트박스 — 현재 시설 갤러리 이미지(facility_01~23 등 10장)에 hover 효과(cursor:pointer)는 있는데, 클릭해도 아무 반응이 없습니다. 클릭하면 이미지가 크게 보이는 라이트박스 기능이 빠져 있습니다.
입지 현황 지도 이미지 — 거금도 내 코리아파크 위치를 보여주는 간략한 위치 안내 이미지(고흥군 지도에 핀 표시)가 있으면 낯선 지역에 처음 오는 방문자에게 도움이 됩니다.


🚁 9대 프로그램 섹션
현재: 9개 카드 중 3개에만 이미지 있음 (드론교육, 드론낚시, 레이저사격). 나머지 6개(다문화교육, 캠핑·글램핑, 트레킹·관광, 특산물, 외국인인프라, 대학연계)는 이모지만 있음
추가하면 좋을 것들

이미지 없는 6개 카드에 각 대표 이미지 — 이미지가 있는 카드와 없는 카드가 섞여 있어 시각적으로 통일감이 없습니다. 특히 '캠핑', '트레킹', '특산물' 카드는 사진이 있으면 훨씬 매력적입니다.
각 카드에 "자세히 보기" 링크 — 카드를 클릭하면 해당 세부 섹션으로 스크롤되거나, 별도 상세 페이지로 이동하는 링크가 있으면 탐색 흐름이 자연스러워집니다. 현재는 카드가 클릭 불가능한 정보 전달용입니다.
드론 자격증 취득 절차 링크 — 드론 교육 카드에 국토교통부 자격증 안내 페이지나 교육 일정 신청 링크가 있으면 관심 있는 방문자의 전환율이 높아집니다.


⛺ 캠핑·숙박 섹션
현재: 캠핑사이트 사진 1장, 오토캠핑/차박/글램핑 설명 텍스트, 링크 없음
추가하면 좋을 것들

각 유형별 사진 — 오토캠핑, 차박, 글램핑 각각의 실제 모습 사진이 있으면 방문자가 어떤 타입을 선택할지 판단하기 쉽습니다. 특히 글램핑 텐트 내부 사진은 필수입니다.
야경·별밤 사진 — 고흥 바다와 별을 담은 캠핑 야경 이미지는 감성적인 방문 동기를 자극합니다.
예약 또는 사전 신청 링크 — 오픈 준비 중이더라도 예약 대기 등록 버튼이 있으면 관심 고객을 놓치지 않을 수 있습니다.
사이트 요금 또는 안내 예시 — 가격대 정보가 전혀 없어 방문자가 예산 계획을 세울 수 없습니다.


⛰ 트레킹 섹션
현재: 트레킹 데크길 사진 1장 + 제주 카페리 사진 1장, 7개 코스 테이블, 링크 없음

각 코스별 대표 사진 — 테이블로 7개 코스를 나열하고 있는데, 사진이 없어서 어떤 풍경인지 상상이 되지 않습니다. 적대봉 정상 조망, 미르마루길 데크, 우도 레인보우교 일몰 등 각 코스 사진이 시각적 동기를 만듭니다.
네이버 지도 또는 등산 앱 링크 — 각 코스명에 네이버 지도, 카카오맵, 또는 두모악·산림청 등산 앱 링크를 연결하면 방문자가 경로를 미리 확인할 수 있습니다.
카페리 예약 링크 — 남해고속 1522-2066 전화번호만 있고, 온라인 예약 링크가 없습니다. 남해고속 공식 예약 페이지 링크를 추가하면 편의성이 크게 올라갑니다.
계절별 추천 코스 사진 — 봄 쑥섬, 여름 익금해수욕장, 가을 단풍 등 계절 이미지가 있으면 재방문 동기를 만들어줍니다.


🎣 드론 낚시 섹션

드론 낚시 실제 장면 동영상 또는 GIF — 텍스트로 아무리 설명해도 "드론이 낚시를?"이라는 생소한 개념을 이미지·영상으로 보여주는 게 훨씬 효과적입니다. 유튜브 영상이 있다면 섹션 내 임베드를 강력히 권장합니다.
방파제 포인트 위치 사진·지도 — 신흥 방파제, 풍남 방파제 위치를 지도나 항공 사진으로 보여주면 낚시 경험자가 포인트를 직관적으로 이해할 수 있습니다.
체험 가격 및 예약 링크 — 드론 대여 비용, 체험 시간 등 구체적인 정보와 예약 방법 링크가 없습니다.


🎯 레이저사격 섹션

드론+레이저 서바이벌 게임 영상 또는 사진 — 독특한 콘텐츠인데 텍스트로만 설명되어 있어 아쉽습니다.
단체 예약 문의 링크 — 팀빌딩, 청소년 교육 용도로 사용 가능한데 단체 신청 경로가 없습니다.

다국어 안내 이미지 또는 링크 — 외국인 근로자 대상 섹션인데 한국어로만 작성되어 있습니다. 영어·베트남어·태국어 등 주요 언어 안내 페이지나 다국어 리플릿 PDF 링크가 있으면 실제 타겟에게 유용합니다.

수확 체험 예약 링크 — 수확 체험 시기(4~6월 양파, 9~11월 석류, 11~12월 유자)가 명시되어 있는데, 체험 신청 링크가 없습니다.

KorPark 전체 개선 제안 — 최종 통합 정리
🔴 즉시 수정해야 할 것 (신뢰·기능 문제)
소개 섹션 이미지 정비가 가장 시급합니다. 왼쪽 AI 렌더링 이미지에 "리모델링 예정 조감도 (AI 렌더링)" 캡션을 반드시 달아야 합니다. 출처 표기 없이 쓰면 방문자가 실제 완성된 공간으로 오인하거나, 가짜 사진으로 의심할 수 있습니다. 배치 순서를 현재(Before) → 미래(After) 순으로 바꾸고, 각각에 Before / After(예정) 레이블을 추가하는 것이 자연스럽습니다. 오른쪽 현재 사진도 고해상도로 교체가 필요합니다.
갤러리 라이트박스 기능이 없는 것도 즉시 수정이 필요합니다. 사이트 내 총 10장의 이미지(소개 2장 + 갤러리 6장 + 레이저사격 2장)에 CSS cursor: pointer가 적용되어 클릭을 유도하고 있지만, 실제로 클릭해도 아무 반응이 없습니다. 클릭 이벤트도 없고 라이트박스 JS도 없습니다. 방문자 신뢰를 직접적으로 깎는 버그입니다.
🟠 이미지 콘텐츠 보강 (전환율 영향)
9대 프로그램 카드 9개 중 이미지가 없는 6개(다문화 교육, 캠핑·글램핑, 트레킹·관광, 특산물 F&B, 외국인 생활 인프라, 대학 연계)에 사진이 필요합니다. 현재 이미지 있는 카드와 이모지만 있는 카드가 뒤섞여 있어 시각적 통일감이 없습니다.
패키지 6개 카드 전부 이미지가 없습니다. 사이트에서 가장 취약한 섹션으로, 구매 전환이 일어나야 하는 곳에 시각적 매력이 전혀 없습니다.
캠핑 섹션은 오토캠핑/차박/글램핑 3가지 유형을 설명하는데 사진이 1장뿐입니다. 특히 글램핑 내부 사진, 야경·별밤 사진이 없습니다.
SJ-ARC 섹션과 문의 섹션에 이미지가 전무합니다. SJ-ARC는 드론 동아리 활동 사진, 문의 섹션은 대표자 프로필 사진이 있으면 신뢰도가 크게 올라갑니다.
외국인 인프라 섹션은 파트너사 5곳 링크는 있으나 로고 이미지가 없어 텍스트 링크뿐입니다.
🟡 링크·기능 보강
사이트 전체에 SNS 링크가 단 하나도 없습니다. 인스타그램, 유튜브, 블로그, 카카오채널 중 운영 중인 채널을 Hero와 푸터에 추가해야 합니다.
드론 낚시 섹션에 유튜브 영상 임베드가 없습니다. "전국 유일"이라고 강조하는 차별화 콘텐츠인데 텍스트와 사진 1장으로만 설명하고 있어 설득력이 약합니다.
카페리 전화번호 1522-2066이 텍스트로만 되어 있고 tel: 링크가 아닙니다. 고흥군청·관광안내소 번호는 이미 tel: 처리가 되어 있는데 이것만 빠져 있습니다.
패키지 6개 전부 "추후 안내"만 표시되어 있어 방문자가 이탈합니다. 예상 가격대나 "관심 등록" 버튼이라도 있으면 훨씬 낫습니다.
