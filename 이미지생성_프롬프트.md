# korpark.com 이미지 생성 프롬프트 (claude.ai용)

아래 프롬프트를 claude.ai에서 하나씩 입력하여 이미지를 생성하세요.
모든 이미지는 **가로 1920x1080 비율**, **한국 남해안 분위기**, **블루 톤** 기조입니다.

## 사용 방법
1. claude.ai에서 위 프롬프트를 하나씩 입력
2. 생성된 이미지를 다운로드
3. 파일명을 괄호 안 이름으로 저장 (예: hero-bg.jpg)
4. `E:\koreapark\korpark-website\images\` 폴더에 저장
5. 이후 index.html에 이미지 반영
